# «Финансовый Планер»

Приложение для учёта личных финансов. 
Предназначено для пользователя, который хочет контролировать свои доходы и расходы, анализировать финансовые привычки и планировать бюджет.

Приложение позволяет:
* регистрировать финансовые операции (доходы и расходы);
* хранить данные о сумме, категории, дате и комментарии операции;
* просматривать и анализировать финансовые операции;
* импортировать и экспортировать данные в формат CSV;
* визуализировать финансовые данные (распределения доходов и расходов по категориям, динамика доходов и расходов во времени)

## Структура проекта:
* `main.py` —  главный файл, с которого начинается программа
* `models.py` —  описание данных (операции, категории и т.п.)
* `storage.py` —  сохранение и загрузка данных
* `gui.py` —  интерфейс программы
* `analysis.py` —  анализ и графики
* `utils.py` —  вспомогательные функции
* `data/` —  файлы с данными
* `docs/` — файлы документации
* `tests/` — файлы тестов



## Примененные библиотеки

Базовые инструменты управления реализованы через графический интерфейс с помощью библиотеки [tkinter](https://docs.python.org/3/library/tkinter.html).

Для анализа использована библиотека [pandas](https://pandas.pydata.org).

Результаты анализа отображаются в виде графиков с помощью [matplotlib](https://matplotlib.org).

Для проверки корректности вводимых данных (даты, суммы, текстовые поля), анализа и очистки входных строк применены [регулярные выражения (re)](https://docs.python.org/3/library/re.html).

Ключевые модули (models.py и analysis.py) покрыты тестами с использованием [pytest](https://docs.pytest.org).

Документация сгенерирована с помощью [sphinx](https://www.sphinx-doc.org).


## Установка

- Клонируйте репозиторий:
```bash
git clone https://github.com/ArkJzzz/
```

- Установите зависимости:
```bash
pip3 install -r requirements.txt
```

## Запуск

```bash
python3 main.py

```
